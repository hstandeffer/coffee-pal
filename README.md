# Coffee Pal

Coffee Pal lets coffee lovers explore all kinds of different coffees from all over the world, all in one place. 

## Features

- Search page to discover coffees listed in our database
- Logged-in users can save coffees to their own list
- Filters for roast type, price, and grind type when searching (coming soon)

## Installation

Create a [firebase](https://firebase.google.com/) project, and add in your credentials using the [firebase example template](https://github.com/hstandeffer/coffee-pal/blob/master/src/components/firebase/firebase-example.js).

```
<!-- Install dependencies -->
npm install
```

```
<!-- Run the project -->
npm run start
```
